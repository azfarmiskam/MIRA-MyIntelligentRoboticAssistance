# MIRA - My Intelligent Robotic Assistant
AIML chatbot for robotic

Requirements:

pip3 install python-aiml
pip3 install gtts
pip3 install pyaudio (for macOS - do first: brew install portaudio)
pip3 install mpg321
pip3 install pyttsx3

run:
#for gtts as tts
python3 chatbot.py

#for pyttsx3 as tts
python3 MIRA_pyttsx3.py
